Certificate Generator Project
=============================

This folder contains the core logic for your Certificate Generator prototype.

INSTRUCTIONS BEFORE USE:
------------------------
Because these libraries are large and need to be kept local, you must download them manually into this folder before opening index.html.

1. Download pdf-lib:
   Go to: https://unpkg.com/pdf-lib/dist/pdf-lib.min.js
   Right-click the page, select "Save As...", and save it in this folder as 'pdf-lib.min.js'.

2. Download fontkit:
   Go to: https://unpkg.com/@pdf-lib/fontkit/dist/fontkit.umd.min.js
   Right-click the page, select "Save As...", and save it in this folder as 'fontkit.umd.min.js'.

3. Run the App:
   Once both JavaScript files are in this folder alongside 'index.html' and 'app.js', simply double-click 'index.html' to open it in your browser and start generating certificates.
